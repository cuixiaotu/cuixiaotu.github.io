Docker学习3

1.docker服务命令

# 启动docker
systemctl start docker 
# 停止docker
systemctl stop docker 
# 查看docker进程状态
systemctl status docker 
# 重启docker
systemctl restart docker 
# 开机重启docker
systemctl enable docker

2.docker 镜像命令

2.1 查看镜像

# 查看本地镜像
docker images

# 查看镜像详细情况
docker inspect ubuntu:18.04

# 查看镜像构建过程
docker history ubuntu:18.04

2.2 搜索镜像

docker search keyword
--filter=is-official=true 
--filter=start=1000

2.3 搜索镜像

# 默认从Docker Hub镜像源下载镜像（配置加速地址） name镜像仓库名称  tag标签（不加会取lastest）
docker pull IMAGE[:TAG]

docker pull ubantu:18.04

# 直接运行
docker run -it ubuntu:18.04

2.4 删除镜像

docker rmi IMAGE[:TAG]
-f, -force：强制删除镜像，即使有容器依赖它
-no-prune：不要清理未带标签的父镜像

3.docker 容器命令

3.1 查看容器

# 查看正在运行的容器
docker ps
# 查看所有容器
docker ps -a

3.2 创建启动容器

docker run -it IMAGE[:TAG] COMMAND
-d 后台执行
-i 保持一直运行
-t 分配终端
--name='' 起名

3.3 进入容器

# 后台运行程序同s进入程序
docker exec -it IMAGE[:TAG] COMMAND
-d 后台执行
-i 保持一直运行
-t 分配终端
--name='' 起名

# 后台先运行程序 再进入程序
docker exec -id IMAGE[:TAG]
docker exec -it name/IMAGE[:TAG] COMMAND

3.4 启动容器

docker run 容器名/容器ID

3.5 删除容器

docker rm 容器名/容器ID

3.6 查看容器详情

docker inspect 容器名/容器ID

其他参数


1. -f, --filter filter：过滤输出内容；
2. --format string：格式化输出内容；
3. --limit int：限制输出结果个数，默认为25个；
4. --no-trunc：不截断输出结果。

二叉查找树

在树的任意一个节点，其左子树的每个节点的值都要小于这个节点的值，而右子树节点的值都要大于这个节点的值。（基于二分查找，满二叉查找数）

public class BinarySearchTree {
  private Node tree;

  public Node find(int data) {
    Node p = tree;
    while (p != null) {
      if (data < p.data) p = p.left;
      else if (data > p.data) p = p.right;
      else return p;
    }
    return null;
  }

  public static class Node {
    private int data;
    private Node left;
    private Node right;

    public Node(int data) {
      this.data = data;
    }
  }
}



public void insert(int data) {
  if (tree == null) {
    tree = new Node(data);
    return;
  }

  Node p = tree;
  while (p != null) {
    if (data > p.data) {
      if (p.right == null) {
        p.right = new Node(data);
        return;
      }
      p = p.right;
    } else { // data < p.data
      if (p.left == null) {
        p.left = new Node(data);
        return;
      }
      p = p.left;
    }
  }
}



public void delete(int data) {
  Node p = tree; // p指向要删除的节点，初始化指向根节点
  Node pp = null; // pp记录的是p的父节点
  while (p != null && p.data != data) {
    pp = p;
    if (data > p.data) p = p.right;
    else p = p.left;
  }
  if (p == null) return; // 没有找到

  // 要删除的节点有两个子节点
  if (p.left != null && p.right != null) { // 查找右子树中最小节点
    Node minP = p.right;
    Node minPP = p; // minPP表示minP的父节点
    while (minP.left != null) {
      minPP = minP;
      minP = minP.left;
    }
    p.data = minP.data; // 将minP的数据替换到p中
    p = minP; // 下面就变成了删除minP了
    pp = minPP;
  }

  // 删除节点是叶子节点或者仅有一个子节点
  Node child; // p的子节点
  if (p.left != null) child = p.left;
  else if (p.right != null) child = p.right;
  else child = null;

  if (pp == null) tree = child; // 删除的是根节点
  else if (pp.left == p) pp.left = child;
  else pp.right = child;
}

2-3查找树

2-3查找树 ： 一棵空树或由2-结点和3-结点组成的树
2-结点：含一个键和两个链接，左链接指向的2-3树中的结点都小于该结点，右链接指向的2-3树中的结点都大于该结点s
3-结点：含两个键和三个链接，左链接指向的2-3树中的结点都小于该结点，右链接指向的2-3树中的结点都大于该结点，中结点指向的结点的键位于该结点两个键的中间

查找

从根节点开始，若和节点中的

插入

1） 如果是空树，创建一个节点并存入值
2） 否则找到叶子节点
3） 如果叶子节点只有一个值，新值也存入该节点
4） 如果叶子节点超过两个值，分离该节点，并把三个值的中位值提升至父节点
5） 如果父节点有三个值，继续4的操作 直至根节点

红黑树

1) 根节点是黑色的
2) 每个节点非黑既红
2) 每个叶子节点都是黑色的空节点（nil）
3) 任何相邻的节点都不能同时为红色
4) 每个节点 到达其可达的叶子节点的所有路径都包含相同数目的黑色节点
